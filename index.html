Jak implementovat kód hlavičky

Kam vložit kód hlavičky?

Kód hlavičky vložte mezi tagy <head> a </head> vaší HTML stránky. Tento kód načítá CSS styly a JavaScript potřebné pro správnou funkci widgetu.

<!DOCTYPE html> <html lang="cs"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Váš e-shop - Výběr dopravy</title> <!-- Sem vložte kód hlavičky PPL widgetu --> <link rel="stylesheet" href="https://www.ppl.cz/sources/map/main.css"> <script type="text/javascript" src="https://www.ppl.cz/sources/map/main.js" async></script> <!-- Další vaše styly a skripty --> </head> <body> ...
TIP: Kód hlavičky stačí vložit pouze jednou, i když máte na stránce více widgetů.

Příklad implementace v redakčním systému

Pokud používáte redakční systém (např. WordPress), obvykle můžete vložit kód hlavičky do nastavení:

  • WordPress: Vzhled > Editor motivu > header.php
  • Shoptet: Administrace > Vzhled a obsah > HTML kód > HTML kód v hlavičce
  • PrestaShop: Vzhled > Záhlaví

Jak implementovat kód těla

Kam vložit kód těla?

Kód těla vložte na místo, kde chcete zobrazit PPL widget. Typicky to bývá v sekci pro výběr dopravy při pokladně e-shopu nebo na stránce s kontakty.

<!-- Příklad v košíku e-shopu --> <div class="shipping-options"> <h3>Vyberte způsob dopravy</h3> <div class="shipping-option"> <input type="radio" id="ppl-parcelshop" name="shipping" value="ppl"> <label for="ppl-parcelshop">PPL Výdejní místo</label> </div> <!-- Kontejner pro mapu s definovanou výškou --> <div class="ppl-map-container" style="height: 600px; margin: 20px 0;"> <!-- Sem vložte kód těla PPL widgetu --> <div id="ppl-parcelshop-map" data-country="cz" data-countries="cz, sk" data-language="cs" data-mode="default"></div> </div> </div>
DŮLEŽITÉ: Rodičovský element (div), do kterého vkládáte PPL widget, by měl mít definovanou výšku (minimálně 550px) a šířku (minimálně 320px) pro správné zobrazení mapy.

Možnosti zobrazení

Widget můžete zobrazit jako:

  • Přímo na stránce - jak je ukázáno výše
  • V modálním okně - které se otevře po kliknutí na tlačítko
  • Na samostatné stránce - například na stránce "Výdejní místa"
TIP: Widget můžete skrýt a zobrazit podle potřeby pomocí JavaScriptu. Například když uživatel vybere "PPL dopravu", můžete zobrazit mapu s výdejními místy.

Jak implementovat JavaScript kód

Kam vložit JavaScript kód?

JavaScript kód vložte na konec těla stránky těsně před uzavírací tag </body>. Tento kód zachytává událost, když uživatel vybere výdejní místo na mapě.

<!-- Obsah stránky --> <!-- Ostatní skripty --> <!-- Sem vložte JavaScript kód pro zachycení vybraného místa --> <script> window.addEventListener('message', function(event) { if (event.data && event.data.event === 'pplPickupPointSelected') { const selectedPoint = event.data.point; console.log('Vybráno výdejní místo:', selectedPoint); // Zde můžete přidat vlastní kód pro zpracování vybraného místa // Například: // document.getElementById('shipping-pickup-id').value = selectedPoint.id; // document.getElementById('shipping-pickup-name').value = selectedPoint.name; // document.getElementById('shipping-pickup-address').value = selectedPoint.address; } }); </script> </body> </html>
POZNÁMKA: Tento kód je potřeba pouze pro widgety v režimu "default", kde chcete zachytit informace o vybraném výdejním místě. Pro režimy "static" nebo "catalog" tento kód nepotřebujete.

Co obsahuje vybrané výdejní místo?

Objekt selectedPoint obsahuje následující informace o vybraném výdejním místě:

  • id - unikátní identifikátor výdejního místa
  • name - název výdejního místa
  • type - typ místa (ParcelShop, ParcelBox, AlzaBox)
  • street - ulice s číslem popisným
  • city - město
  • zip - PSČ
  • country - země (cz, sk, pl, de)
  • latitude - zeměpisná šířka
  • longitude - zeměpisná délka
  • a další informace jako otevírací doba, služby, atd.
TIP: Pro ladění si můžete zobrazit kompletní data výdejního místa pomocí console.log(selectedPoint) a prohlédnout si všechny dostupné informace.

Příklad praktického použití

<script> window.addEventListener('message', function(event) { if (event.data && event.data.event === 'pplPickupPointSelected') { const selectedPoint = event.data.point; console.log('Message listener - Vybráno výdejní místo:', selectedPoint); } // Uložení dat do skrytých polí formuláře document.getElementById('pickup_id').value = selectedPoint.id; document.getElementById('pickup_name').value = selectedPoint.name; document.getElementById('pickup_address').value = selectedPoint.street + ', ' + selectedPoint.zip + ' ' + selectedPoint.city; // Zobrazení informace o vybraném místě uživateli const infoContainer = document.getElementById('selected-point-info'); infoContainer.innerHTML = ` <div class="selected-point"> <h4>Vybrané výdejní místo</h4> <p><strong>${selectedPoint.name}</strong></p> <p>${selectedPoint.street}, ${selectedPoint.zip} ${selectedPoint.city}</p> </div> `; // Povolení tlačítka pro pokračování document.getElementById('continue-button').disabled = false; } }); </script>

How to implement the header code

Where to insert the header code?

Insert the header code between the <head> and </head> tags of your HTML page. This code loads the CSS styles and JavaScript needed for the widget to function properly.

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Your e-shop - Shipping selection</title> <!-- Insert PPL widget header code here --> <link rel="stylesheet" href="https://www.ppl.cz/sources/map/main.css"> <script type="text/javascript" src="https://www.ppl.cz/sources/map/main.js" async></script> <!-- Your other styles and scripts --> </head> <body> ...
TIP: The header code only needs to be inserted once, even if you have multiple widgets on the page.

Example implementation in content management systems

If you're using a content management system (e.g., WordPress), you can usually insert the header code in the settings:

  • WordPress: Appearance > Theme Editor > header.php
  • Shoptet: Administration > Appearance and content > HTML code > HTML code in header
  • PrestaShop: Appearance > Header

How to implement the body code

Where to insert the body code?

Insert the body code at the location where you want to display the PPL widget. This is typically in the shipping selection section at the e-shop checkout or on the contact page.

<!-- Example in e-shop cart --> <div class="shipping-options"> <h3>Select shipping method</h3> <div class="shipping-option"> <input type="radio" id="ppl-parcelshop" name="shipping" value="ppl"> <label for="ppl-parcelshop">PPL Pickup point</label> </div> <!-- Container for map with defined height --> <div class="ppl-map-container" style="height: 600px; margin: 20px 0;"> <!-- Insert PPL widget body code here --> <div id="ppl-parcelshop-map" data-country="cz" data-countries="cz, sk" data-language="en" data-mode="default"></div> </div> </div>
IMPORTANT: The parent element (div) into which you place the PPL widget should have a defined height (minimum 550px) and width (minimum 320px) for proper map display.

Display options

You can display the widget as:

  • Directly on the page - as shown above
  • In a modal window - which opens after clicking a button
  • On a separate page - for example on a "Pickup points" page
TIP: You can hide and show the widget as needed using JavaScript. For example, when a user selects "PPL delivery", you can display the map with pickup points.

How to implement the JavaScript code

Where to insert the JavaScript code?

Insert the JavaScript code at the end of the page body just before the closing </body> tag. This code captures the event when a user selects a pickup point on the map.

<!-- Page content --> <!-- Other scripts --> <!-- Insert JavaScript code for capturing the selected point here --> <script> window.addEventListener('message', function(event) { if (event.data && event.data.event === 'pplPickupPointSelected') { const selectedPoint = event.data.point; console.log('Selected pickup point:', selectedPoint); // Here you can add your own code for processing the selected point // For example: // document.getElementById('shipping-pickup-id').value = selectedPoint.id; // document.getElementById('shipping-pickup-name').value = selectedPoint.name; // document.getElementById('shipping-pickup-address').value = selectedPoint.address; } }); </script> </body> </html>
NOTE: This code is only needed for widgets in "default" mode, where you want to capture information about the selected pickup point. For "static" or "catalog" modes, you don't need this code.

What does the selected pickup point contain?

The selectedPoint object contains the following information about the selected pickup point:

  • id - unique identifier of the pickup point
  • name - name of the pickup point
  • type - type of location (ParcelShop, ParcelBox, AlzaBox)
  • street - street with house number
  • city - city
  • zip - ZIP code
  • country - country (cz, sk, pl, de)
  • latitude - latitude
  • longitude - longitude
  • and other information such as opening hours, services, etc.
TIP: For debugging, you can display the complete data of the pickup point using console.log(selectedPoint) and view all available information.

Example of practical use

<script> window.addEventListener('message', function(event) { if (event.data && event.data.event === 'pplPickupPointSelected') { const selectedPoint = event.data.point; // Saving data to hidden form fields document.getElementById('pickup_id').value = selectedPoint.id; document.getElementById('pickup_name').value = selectedPoint.name; document.getElementById('pickup_address').value = selectedPoint.street + ', ' + selectedPoint.zip + ' ' + selectedPoint.city; // Displaying information about the selected point to the user const infoContainer = document.getElementById('selected-point-info'); infoContainer.innerHTML = ` <div class="selected-point"> <h4>Selected pickup point</h4> <p><strong>${selectedPoint.name}</strong></p> <p>${selectedPoint.street}, ${selectedPoint.zip} ${selectedPoint.city}</p> </div> `; // Enabling the continue button document.getElementById('continue-button').disabled = false; } }); </script>
